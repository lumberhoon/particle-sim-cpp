# Particle Sim C++

Early C++ particle simulation practice project.

## Current features
- Terminal-based 2D particle movement
- Wall bouncing
- Vector-based particle storage
- Random particle generation
- Basic command router system

## Planned features
- Raylib visualization
- Better input validation
- CPU benchmarking
- AoS vs SoA comparison
- CUDA version
- ROCm/HIP exploration later
